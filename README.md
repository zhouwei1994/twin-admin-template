# twin-layout 双生布局管理后台解决方案
1. 传统的管理后台布局
2. 桌面式管理后台布局

### 项目运行

1. 下载项目
```
git clone git@github.com:zhouwei1994/twin-admin-template.git
```
2. 进入到项目的目录
```
cd twin-admin-template
```
3. 下载npm依赖
```
yarn 或 npm install
```
4. 运行项目
```
npm run dev
```
